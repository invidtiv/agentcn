## July 2026 - LangGraph Support

We added 19 LangGraph agent recipes built with Dawn as the meta-framework, bringing the catalog to 76 recipes across four frameworks. LangGraph recipes use Dawn's file convention (`index.ts` + `instructions.md` + `tools/*.ts`).

## July 2026 - Mastra Framework Support

We added 19 Mastra agent recipes to the registry, bringing the catalog to 57 recipes across three frameworks. Mastra recipes use the file-based agent convention (`config.ts` + `instructions.md` + `tools/*.ts`).

## June 2026 - Initial Release

We shipped the initial release of agentcn with 19 agent recipes for both Eve and Flue.
