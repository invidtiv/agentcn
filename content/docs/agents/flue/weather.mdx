## Summary

The **Weather Agent** is the classic starter: ask about a place and it reports the
current conditions. It geocodes the location and fetches live weather from the
free Open-Meteo API — no key required. Reach for it as a minimal, end-to-end
example of an agent with a single tool.

## Install

```bash
npx shadcn@latest add @agentcn/flue/weather
```

## Composition

```text
agents/
└── weather.ts          # Agent with the get_weather tool
tools/
└── get-weather.ts      # Geocode + current forecast (Open-Meteo)
```

## Customization

- **Add a forecast.** Extend `get-weather.ts` to request daily/hourly data.
- **Add units.** Pass a `units` parameter (metric/imperial) to the API.
- **Add a workflow.** Wrap it in a workflow with a typed forecast schema.
- **Swap the model.** Change the `model` field on the agent.
