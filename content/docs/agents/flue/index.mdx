[Flue](https://flueframework.com) is an agent runtime with first-class
subagents, skills, and bounded workflows. These recipes install the agent,
workflow, tool, and skill files straight into your Flue project with the shadcn
CLI.

## Available agents

Browse the full catalog in the [Agents](/docs/agents) section — every recipe is
available for Flue. A good starting point is the
[Deep Search](/docs/agents/flue/deep-search) agent.

Each recipe page has a live preview, a summary, the install command, the file
composition it adds, and customization notes. See
[Installation](/docs/installation) for the one-time project setup.
