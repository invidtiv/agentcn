Agent recipes are complete, copy-paste backend patterns for the
[Eve](https://vercel.com/docs/eve) and [Flue](https://flueframework.com)
frameworks. Each recipe ships its instructions, tools, skills, and (for Flue)
workflows, installable with the shadcn CLI.

Recipes are organized by framework — pick the section that matches the runtime
you already use:

- **[Eve agents](/docs/agents/eve)** — durable, filesystem-first agents that run
  on Vercel Functions.
- **[Flue agents](/docs/agents/flue)** — agents with first-class subagents,
  skills, and bounded workflows.

## Available agents

- **Competitor Intelligence** — scrapes and analyzes competitor pages, then
  returns a structured intelligence report. Available for
  [Eve](/docs/agents/eve/competitor-intel) and
  [Flue](/docs/agents/flue/competitor-intel).

More recipes coming soon. Each one includes an install command, prerequisites, a
live preview, and a breakdown of every file it installs.
