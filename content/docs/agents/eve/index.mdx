[Eve](https://vercel.com/docs/eve) is a filesystem-first framework for durable
backend AI agents that run on Vercel Functions. These recipes install the agent
files (instructions, tools, and skills) straight into your Eve project's
`agent/` directory with the shadcn CLI.

## Available agents

Browse the full catalog in the [Agents](/docs/agents) section — every recipe is
available for Eve. A good starting point is the
[Deep Search](/docs/agents/eve/deep-search) agent.

Each recipe page has a live preview, a summary, the install command, the file
composition it adds, and customization notes. See
[Installation](/docs/installation) for the one-time project setup.
